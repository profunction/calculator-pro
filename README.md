# Calculator Pro

A free, multi-language business calculator website with 10 calculators. Supports English, 华文, and Bahasa Melayu. Country-specific algorithms for Malaysia (RM), China (CNY), US (USD), and more.

## Live Demo
👉 https://profunction.github.io/calculator-pro

## Calculators
1. Scientific Calculator
2. Loan Calculator (with amortization)
3. Mortgage Calculator (Malaysia BLR support)
4. Investment Calculator (inflation-adjusted)
5. VAT/Tax Calculator (Malaysia SST, China VAT, Singapore GST)
6. Salary Calculator (Malaysia EPF/SOCSO/EIS)
7. Car Lease Calculator (Malaysia road tax)
8. Profit Margin Calculator
9. Rental Yield Calculator (Malaysia RPGT)
10. Equipment Lease Calculator (Malaysia Section 46A)

## Features
- 3 languages: English, 华文 (Chinese), Bahasa Melayu
- Currency selector: RM, ¥, $, S$, £, €, ฿, ₫, ₱
- Country-specific calculation algorithms
- Dark mode
- Mobile responsive
- Google Sign-In

## Tech Stack
Pure HTML + CSS + JavaScript — no build step, no dependencies.
